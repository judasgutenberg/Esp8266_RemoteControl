These are files specifically to flash onto an ESP8266 microcontroller using 
the Arduino IDE or a functional equivalent. You will need to customize config.cpp with info about your server, sensor attachments, and WiFi credentials. Many of the items in there can be left unchanged until you need deep customization.

At some point the goal with these files is to compile them automatically to a .bin file using the existing server-based information about the specific devices to produce smaller customized flash images that can run on a variety of architectures (not just ESP8266).
